Masked Bandits
Website development

Masked bandits is a python game which works via terminal input. I made the game with Brittain, Andre and Ryan from the Code Nation develop course.

The game is a GTA style Heist with a score board. Each level is a different location where, the player have 3 strategies to choose from. Each strategy ranges in difficulty and also has the ability to fail.
